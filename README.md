## Graphify Output (`graphify-out/`)

The `graphify-out/` folder was generated by **Graphify** — a codebase intelligence tool that scans a repository and produces an interactive force-directed dependency graph.

- **graph.json** — Machine-readable knowledge graph used by AI agents for token-efficient codebase navigation (~40× fewer tokens per targeted query vs. full reads).
- **graph.html** — Standalone interactive HTML visualisation of the dependency network.
- **GRAPH_REPORT.md** — Natural-language summary of nodes, edges, communities, and key dependency paths.

Stats for this website: **29 nodes · 38 edges · 4 communities**

## Deployment

This site is deployed as a static site on [Vercel](https://vercel.com).

## Local Development

Serve with any static file server, for example:

```bash
python3 -m http.server 3000
```

Then open [http://localhost:3000](http://localhost:3000).
