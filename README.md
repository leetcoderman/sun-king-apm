# Sun King APM — Portfolio Website

A personal portfolio website built as part of the Sun King hiring assessment.

## Structure

```
website/
├── index.html        # Main single-page application
├── styles.css        # Global styles
├── script.js         # Interactive behaviours & animations
├── assets/
│   ├── profile-pic.jpg
│   ├── Shikhar_Gupta_Resume.pdf
│   └── images/
└── vercel.json       # Vercel static deployment config
```

## Deployment

This site is deployed as a static site on [Vercel](https://vercel.com).

## Local Development

Serve with any static file server, for example:

```bash
python3 -m http.server 3000
```

Then open [http://localhost:3000](http://localhost:3000).
